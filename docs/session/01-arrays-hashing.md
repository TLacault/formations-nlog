# Arrays & Hashing

## Arrays

```cpp title="C++ Base Template"
#include <iostream>

using namespace std;

int main(int ac, const char **av) {
    cout << "Hello, World!" << endl;

    for (int i = 0; i < ac; ++i) {
        cout << "Argument " << i << ": " << av[i] << endl;
    }

    return EXIT_SUCCESS;
}
```
